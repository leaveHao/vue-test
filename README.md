## 运行
### 服务器端
**项目后端服务器使用node.js编写，运行前请确认系统已安装相关应用**
* `npm install` 安装依赖 
* `node main.js` 运行服务

### 客户端
* `npm install` 安装依赖
* `npm run serve` 运行服务
